## Python. Предобработка данных: Обработка исторических данных об играх

### Цель:
- Подготовить данные для дальнейшего анализа развития игровой индустрии.

### Основные шаги:
- загрузка и знакомство с данными 
- проверка наличия ошибок, пропусков и дубликатов
- фильтрация данных
- проведение категоризации данных для более удобного последующего анализа
